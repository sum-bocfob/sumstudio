【Unity packageについて】

インポート方法

zipファイルを解凍します。
Unity上で、Assets -> Import Package -> Custom Packageで、解凍した.unitypackageファイルを選択してください。

シーン、マテリアル

付属のシーンでアニメーションを確認する場合は
Edit -> Project Settings -> Player -> Other Settingsの Active Input Handling を Both にしてください。

付属のマテリアルはBuilt-in Render Pipeline用です。
URP用にするには、Materialフォルダ内のマテリアルを選択して、 Edit -> Rendering -> Materials -> Convert Selected Built-in Materials to URPを選択してください。
